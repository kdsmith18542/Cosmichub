🔗 URL Format Testing

The PHP router supports multiple URL formats:

1. Path Info Style (Recommended):

Home (index.php/) Login (index.php/login) Dashboard (index.php/dashboard)

2. Query String Style (Fallback):

Home (?route=/) Login (?route=/login) Dashboard (?route=/dashboard)

📁 File Structure

Your deployment should have this structure:

public_html/ (your domain's public folder)
├── index.php (PHP router entry point)
├── assets/ (CSS, JS, images)
├── favicon.ico
└── php-routing-test.html (this file)

app/ (application files - outside public_html)
├── libraries/PHPRouter.php
├── controllers/
├── models/
└── routes/web.php
